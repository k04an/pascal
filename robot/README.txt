Простенькая программа-черепашка
На данный момент поддерживается только движение, раскраска и очищение клеток.

Для управления черепашкой необходимо создать текстовый файл.

Для начала записываются данные конфигурации в виде "(параметр) = (знаение)", следуеют обратить внимание
на пробелы до и после "=".

Список параметров
WIDTH - ширина поля в клетках
HEIGHT - высота поля в клетках
X - начальная координата X черепашки
Y - начальная координата Y черепашки
DELAY - задержка между выполнениями действий в мс. (Необязательный параметр. По-умолчанию = 500мс)

После конфигурации поля указывается ключевое слово "START". Затем идут команды:

up - переместиться вверх
down - переместится вниз
left - переместится влево
right - переместится вправо
pendown - "опустить перо". С опущенным пером черепашка оставляет за собой след
penup - "поднять перо"
cleanup - очистить клетку, на которой находится черепашка

В директории с программой присутсвует пример написания файла инструкций. (inst.txt)
